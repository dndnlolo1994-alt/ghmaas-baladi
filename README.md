# غماس بلدي

ويب أب لمطعم غماس بلدي: واجهة زبون عربية حديثة + لوحة أدمن لتعديل الأسعار، الأصناف، الصور، الفروع، ونصوص الصفحة.

## التشغيل المحلي

شغل التطبيق:

```powershell
npm start
```

ثم افتح:

```text
http://localhost:5174
```

لوحة الأدمن:

```text
http://localhost:5174/admin
```

بيانات الدخول تقرأ من متغيرات البيئة `ADMIN_EMAIL` و `ADMIN_PASSWORD`.
لتغيير كلمة المرور وقت التشغيل:

```powershell
$env:ADMIN_PASSWORD='new-password'; npm start
```

## الإدارة والبيانات

- المنيو، الصور، الأسعار، الفروع، وأرقام التواصل قابلة للتعديل من لوحة الأدمن.
- أي تعديل يتم حفظه من الأدمن ينعكس مباشرة على واجهة الزبون.
- `data/site.json` يعمل كنسخة احتياطية محلية عند الحاجة.
