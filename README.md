# غماس بلدي

ويب أب لمطعم غماس بلدي: واجهة زبون عربية حديثة + لوحة أدمن لتعديل الأسعار، الأصناف، الصور، الفروع، ونصوص الصفحة.

## التشغيل المحلي

شغل التطبيق:

```powershell
npm start
```

ثم افتح:

```text
http://localhost:5174
```

لوحة الأدمن:

```text
http://localhost:5174/admin
```

كلمة المرور الافتراضية:

```text
admin123
```

لتغيير كلمة المرور وقت التشغيل:

```powershell
$env:ADMIN_PASSWORD='new-password'; npm start
```

## مصادر البيانات

- المنيو، الصور، الأسعار، رقم التواصل، ورابط الخريطة مأخوذة من: https://amenomenu.com/417
- الفروع الإضافية ظاهرة في حساب إنستغرام غماس بلدي ونتائج البحث العامة.

## ملاحظات قبل النشر

- راجع الأسعار قبل الإطلاق لأن أسعار المطاعم قابلة للتغيير.
- يفضل نقل الصور إلى استضافة المشروع النهائية بدل الاعتماد على روابط Ameno Menu الخارجية.
- النص الموجود في آخر الصفحة: `إهداء من المسار الذهبي بسبب وقتهم من النشامى`.
- بيانات الأدمن تحفظ في `data/site.json`.
